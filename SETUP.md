# TaskFlow - Delightful Task Management

## Quick Start

- git clone https://github.com/DanielcB123/taskflow.git
- cd taskflow

- cp .env.example .env
- composer install
- npm install

- php artisan key:generate
- php artisan migrate:fresh --seed

- npm run dev
- php artisan serve
